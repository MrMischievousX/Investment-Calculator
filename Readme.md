## Development

Make Sure you have [React Native development environment setup](https://reactnative.dev/docs/environment-setup)

```bash
# Install dependencies
yarn install
or
npm install
```

```bash
# Run android
npm run android
or
yarn android
```

```bash
# Run ios
npm run ios
or
yarn ios
```

```bash
# Run unit tests
npm run test
or
yarn test
```

```bash
# Check test coverage
npm run coverage
or
yarn coverage
```
